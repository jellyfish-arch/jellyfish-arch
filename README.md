### 🚀 [dev-portfolio-projects](https://github.com/jellyfish-arch/dev-portfolio-projects)
> Polished, real-world projects built to demonstrate applied engineering skills.

#### 🧠 DevTrack AI
A **developer growth intelligence system** — analyzes GitHub profiles or manual input to generate a developer score, skill classification, AI-powered insights (via Ollama/local LLM), and a career roadmap.
- GitHub API integration, smart scoring engine, score history via `localStorage`
- Dark/light theme, animated score display, language breakdown bars
- **Tech:** HTML, CSS, JavaScript

#### ⚙️ NexusFlow — Async Task Orchestrator
A high-performance **async task orchestration engine** built as a showcase of advanced Python.
- `asyncio`-based non-blocking execution with priority-based scheduling (High / Medium / Low)
- Exponential backoff retry logic, JSON state persistence, observer-pattern monitoring
- Full `pytest` test suite with mocking and edge case validation
- **Tech:** Python, asyncio, pytest

#### 🔐 CipherVault — Password Manager & Encryption Toolkit
A **browser-based password manager** powered by real cryptography — **AES-256-GCM encryption** and **PBKDF2 key derivation** via the native Web Crypto API.
- Encrypted vault (localStorage), password generator, text encrypt/decrypt, security audit engine, secure recovery system
- Lock screen UI, dark/light themes, zero dependencies, zero servers — fully private and offline-capable
- **Tech:** HTML, CSS, JavaScript, Web Crypto API

#### 🧪 Polyglot Experiment
A cross-language experiment solving the same problems in Python, Java, and JavaScript — demonstrating multi-language thinking and adaptability.
- **Tech:** Python, Java, HTML/JS
